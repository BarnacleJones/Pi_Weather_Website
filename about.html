About this project

Briefly details the steps to build this project

TLDR; Tech stack used

To briefly summarise, the following technologies were used in this project:

  • Python
  • HTML/CSS/Javascript
  • Apache/PHP/MySQL
  • PHPMyAdmin
  • Bootstrap
  • NPM (for date-and-time library).
  • OpenWeather API for current forecase information.
  • Raspberry Pi + Sense HAT
  • GitHub repo

Thanks for checking this out! Read a bit more about the steps involved below.

Using the Sense HAT

I first started tinkering with the sense hat API, blinking lights and gathering readings via python scripts.

  • See API Documentation Below
  • Sense HAT API Reference

Challenge here was getting into writing Python scripts for the first time.
Really came to enjoy the regimented structure of Python!

Writing temperature/humidiy/pressure readings to CSV

My next step was writing the sensory readings to a CSV file.

  • See tutorial I followed below
  • Writing out to CSV tutorial

This was a great tutorial, one of the rare ones where everything seemed to work the first try!

Setting up Apache/PHP on the Raspberry Pi

The next step was setting up a server so I could display readings on a simple website locally.

  • Used a number of sources to get this up and running
  • Overall straightforward.

The only challenge here was diving into the unknown! I was suprised at the simplicity of getting an Apache server up and running.

Pivoting to MySQL

I decided the CSV file would be a bit simplistic, so took on the challenge of installing mySQL and PHPAdmin on the Apache server

  • This was a lengthy step as I needed to figure out how to establish a connection with Python
  • Also needed to sharpen my PHP knowledge to get the connection to the database.
  • Also needed to design the database and get that going on my Apache server!

This was by far the most challenging portion of the project! I have definitely sharpened my PHP debugging skills.
I also was using Github to push from my development environment + pull on the server. This is why this GitHub repository has hundreds of commits!

Completed project!

Final touches.

  • Bootstrap/CSS/PHP/HTML
  • Node used for date-and-time library
  • OpenWeather API used for current forcasted data

Now all the pieces were working I made a friendly website. Next step - hosting on the actual web via the Pi!
